Help

Qualification

Opens on each job page. Press Ctrl+Shift+Q (or click Build Prompt & Copy) to scrape the job and copy a ready-made AI prompt to your clipboard.

  • Paste into ChatGPT, Claude, or any AI chat.
  • The AI rates fit (High / Medium / Low) and says whether to apply.
  • Edit the Prompt Template to change how the AI evaluates jobs.
  • Paste your CV once — it's saved and reused automatically.

Capture Tabs

Scrapes all open job tabs and exports a JSON array:

[ { date, url, title, company, description }, … ]

Use Copy JSON or Download JSON to save the result. Great for building a shortlist or tracking applications.

⚠️ Before running Capture: every job tab must be fully loaded. Opening a link in a new tab is not enough — you need to click each tab so it gets focus and the page content fully renders. Check this manually before hitting Scan.

Keyboard Shortcut

Ctrl+Shift+Q works even when the popup is closed — as long as a job tab is open.

To change the shortcut: go to chrome://extensions/shortcuts.

Tips

  • The extension only reads tabs on linkedin.com.
  • Your CV and template are stored locally — nothing is sent anywhere.
  • If scraping returns empty fields, reload the job page and try again.
